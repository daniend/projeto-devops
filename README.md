# Projeto DevOps com Terraform

Este projeto demonstra conceitos de DevOps utilizando:

- Python
- Testes automatizados
- GitHub Actions
- Terraform

## Aplicação

Função simples em Python que soma dois números.

## Testes

Executados automaticamente utilizando pytest.

## CI/CD

A pipeline executa automaticamente a cada push no GitHub.

## Infraestrutura como Código

A infraestrutura foi provisionada usando Terraform, criando um bucket S3 na AWS.
